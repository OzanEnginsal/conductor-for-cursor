## Status Definitions

- **Planning**: Storage component is being planned, spec and plan are being created
- **In Progress**: Storage component is currently being implemented
- **Completed**: Storage component is complete and ready for use
- **Blocked**: Storage component is blocked by dependencies or issues
- **Cancelled**: Storage component was cancelled
- **Reverted**: Storage component was reverted
