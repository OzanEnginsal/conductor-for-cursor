# Connectors

## Overview
This file tracks all API connector tracks in the project. Each connector represents a high-level unit of work that has been planned and is being implemented for a specific platform API.

## Connector Status Legend

- **Planning**: Connector is being planned, spec and plan are being created
- **In Progress**: Connector is currently being implemented
- **Completed**: Connector has been fully implemented and verified
- **Blocked**: Connector is blocked and cannot proceed
- **Cancelled**: Connector has been cancelled
- **Reverted**: Connector has been reverted
